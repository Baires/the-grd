Install

bower install thegrd
gem install thegrd

Usage

Define how many columns you need (default is 12)
$grid-columns:12
Define your breakpoint (50em by default)
$medium:"(min-width: 50em)"
Start a new grid:
<div class="row"> <div class="column grid-2-of-12"> Content </div> <div class="column grid-6-of-12"> Content </div> <div class="column grid-8-of-12"> Content </div> </div>
